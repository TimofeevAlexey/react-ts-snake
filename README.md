# React TypeScript Snake

## Для запуска игры:

```
npm i
npm start
```

### Управление

← ↑ → ↓ - движение змейки

пробел - новая игра

### Реализация
* Язык программирования: TypeScript
* Cтек технологий: React
* Bundler: Webpack
* Игру можно начать по нажатию пробела  
* Змейка может проходить сквозь стены (задается на начальном экране)
